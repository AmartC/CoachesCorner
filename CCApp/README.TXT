USER INSTRUCTIONS (How to run):

  1. Run the compile-jar-selfsign.bat file, make sure you have Java installed.
  2. Open the CCApp.html file in Internet Explorer.
  3. Copy the address on top of the browser.
  4. Open Java's control panel (Configure Java).
  5. Go to the security tab.
  6. Add the copied address to the Exception List.
  7. Now reload the CCApp.html page.
  8. The application should be running.
