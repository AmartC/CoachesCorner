USER INSTRUCTIONS (How to run):

  1. Install Java 8 JRE and JDK from the web.
  2. Run the compile-jar-selfsign.bat file.
  3. Open the CCApp.html file in Internet Explorer.
  4. Copy the address on top of the browser.
  5. Open Java's control panel (Configure Java).
  6. Go to the security tab.
  7. Add the copied address to the Exception List, adding file:\\\ to the front of it.
  8. Close your browser, and reopen the .html file.
  9. Accept the prompts to run the Java Applet.
  10. The application should be running.
